Tribal Wars Engine

Ferramentas para o Tribal Wars

Download this project as a .zip file Download this project as a tar.gz file

Tribal Wars Engine é uma ferramenta independente para auxiliar jogadores de Tribal Wars.

Ela conta com diversas funcionalidades:

  • Calculadora de tempo de recrutamento
  • Calculadora de informações de exército
  • Calculadora de pontos e população disponível
  • Caculadora de distância
  • Calculadora de OD
  • Simluador de batalha

Todas as ferramentas se adaptam às configurações do mundo, mostrando apenas as informações relevantes a ele. Por exemplo, mundos sem igreja não terão a opção de religioso no simulador.

Ela possui diferentes configurações para cada mundo do servidor BR, além da capacidade de alterar ou criar novos mundos.

É possível salvar modelos de tropas para maior facilidade de uso das ferramentas.
